# Caiber Website - Python Flask Version

## Setup Instructions

### Step 1: Install Python Dependencies
```
pip install -r requirements.txt
```

### Step 2: Create Folder Structure
Make sure your folder structure looks like this:
```
your-project-folder/
├── app.py
├── requirements.txt
├── templates/
│   └── index.html
```

### Step 3: Run the Flask App
```
python app.py
```

### Step 4: Open in Browser
Go to `http://localhost:5000` in your web browser.

---

## How It Works

- **app.py** - Main Flask server that runs the website
- **templates/index.html** - The website (HTML, CSS, JavaScript)
- **requirements.txt** - Python dependencies

All the buttons work, the modal pops up, form submission is handled, and animations are exactly the same as the React version.

---

## To Stop the Server
Press `CTRL + C` in your terminal.
